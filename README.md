# House Price Prediction (Machine Learning)

## Objective
The goal of this project is to predict house prices using machine learning based on property features such as area, bedrooms, bathrooms, and location.

## Dataset
Zameen Lahore Property Dataset

Features used:
- Location
- Beds
- Baths
- Area_sqft

Target variable:
- Price_thnds

## Models Used
- Linear Regression

## Evaluation Metrics
- Mean Absolute Error (MAE)
- Root Mean Squared Error (RMSE)

## Results
The model was able to predict house prices with reasonable accuracy. Visualization of actual vs predicted prices shows the model performance.

## Technologies
- Python
- Pandas
- Scikit-learn
- Matplotlib
- Seaborn
