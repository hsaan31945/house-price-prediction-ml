AI/ML Engineering Internship - Task 6: House Price PredictionObjectiveThe primary goal of this task is to develop a machine learning model capable of predicting house prices in Lahore, Pakistan, based on property features such as size (square footage), number of bedrooms, number of bathrooms, and location.DatasetSource: Zameen Lahore Dataset (CSV format).Features Used: * Area_sqft: The total area of the property.Beds & Baths: Number of bedrooms and bathrooms.locality_enc: Encoded geographical location.bath_bed_ratio: A derived feature representing the ratio of bathrooms to bedrooms.area_log: A log-transformed version of square footage to normalize data distribution.Models AppliedI implemented and compared two distinct regression models to determine the best fit for real estate data:Linear Regression:Used as a baseline model.Features were scaled using StandardScaler to ensure coefficients were treated fairly.Gradient Boosting Regressor:A more complex ensemble model designed to capture non-linear relationships.Hyperparameters: n_estimators=300, learning_rate=0.08, max_depth=4.Key Results and FindingsThe models were evaluated using standard regression metrics to ensure accuracy and
